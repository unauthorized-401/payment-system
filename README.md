# 카카오페이 손해보험 사전과제
사전과제 1 결제시스템

결제요청을 받아 카드사와 통신하는 인터페이스를 제공하는 결제시스템
- 임베디드 데이터베이스 H2 사용
- 카드결제, 결제취소, 결제정보 조회 REST API 구현
- 단위 테스트로 각 기능 검증
- 선택 문제 부분취소 API 구현 후 테스트 케이스 확인

## 개발 환경
- IntelliJ
- Java 17 + Spring Boot 3.1.4
- H2 1.3.176
